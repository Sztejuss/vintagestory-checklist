Early Survival

  • Gather basic resources (sticks, stones, grass)
  • Craft first tools (knapping flint tools)
  • Build a shelter before nightfall

Basic Crafting

  • Find food sources (berries, mushrooms, hunting)
  • Make a firepit for cooking and warmth
  • Begin clay forming (pottery, storage vessels)

Advanced Tools

  • Start copper prospecting for metal tools

Metal Age

  • Smelt copper and make first metal tools
